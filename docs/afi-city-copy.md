## Framing guardrails (keep copy accurate)

Factory **does**: help define, validate, and prepare AFI-powered strategy and
agent workflows (locally via the CLI, for agents via MCP, for developers via the
library), and provide templates/examples/starters.

Factory **does not**: execute trades, act as a trading terminal, score signals
itself, run provider execution, run the full AFI protocol, build or persist
evidence, or replace Gateway, Machine, Vault, or BUY SELL TERMINAL.

Where Factory sits: **Factory** builds the playbooks · **Machine** scores the
signals · **Gateway** provides protocol access · **MCP** connects agents ·
**BUY SELL TERMINAL** (if/when built) is the trader-facing cockpit.
